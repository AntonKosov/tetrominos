## Controls

`Ctrl-C` - Exit

### Start Screen

`Esc` - Start game

### Game Screen

`Esc`    - Pause

`Left`   - Move left

`Right`  - Move right

`Up`     - Rotate left

`Down`   - Rotate right

`Enter`  - Drop

`Ctrl-C` - Exit

### Pause Screen

`Esc` - Resume game

### Game Over Screen

`Esc` - Start game
